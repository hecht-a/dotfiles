## Window management
| Key | Action |
|--|--|
| `f4` | Split current window |
| `shift + up` | Move window up |
| `shift + left` | Move window left |
| `shift + right` | Move window right |
| `shift + down` | Move window down |
| `cmd + down` | Decrease tab size |
| `cmd + up` | Increase tab size |
